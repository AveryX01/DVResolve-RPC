# [DVResolve-RPC](https://github.com/v4ish/rpc)
Discord Rich presence for [DaVinci Resolve](https://www.blackmagicdesign.com/products/davinciresolve/) written in c#

## Features

- Super lightweight 
- Supported OS: Windows
- Runs on all CPU architecture
